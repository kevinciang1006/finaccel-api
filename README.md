Up # Finaccell Express backend

This is the backend using Express.js

## Installation

Use the package manager [npm](https://nodejs.org/en/download/) to install.

Setup the backend

```bash
npm install
sequelize db:migrate
sequelize db:seed:all
```

## Usage

Start the backend

```bash
npm start
```
